A

Atores - Use Case Diagram in Cosmos Design
Este diagram lista todos os atores que têm casos de uso identificados.

C

CNI - Actor in Cosmos Design
Este ator não participará de qualquer caso de uso referente às atividades dos 4 campos de trabalho. Participará, contudo, de casos de uso referentes à manutenção do sistema, atualização de dados e licenciamento de usuários. Possui esfera de atuação nacional.

Cosmos Design - project

D

DEI - Actor in Cosmos Design
Este ator participará de todos os casos de uso. Não existirão funcionalidades dos sistemas que não poderão ser acessadas por este ator. Por outro lado, existem alguns casos de uso que serão associados apenas a este ator e a nenhum outro. Possui esfera de atuação internacional.

DITM - Actor in Cosmos Design
Direção Internacional do Trabalho da Mocidade. Este ator só participará dos casos de uso relacionados às atividades do Trabalho da Mocidade. Possui esfera de atuação internacional.

DNLR - Actor in Cosmos Design
Este ator participará de todos os casos de uso que envolvam as atividades do Lectorium Rosicrucianum. Possui esfera de atuação nacional.

DNTM - Actor in Cosmos Design
Este ator participará de todos os casos de uso que envolvam as atividades do Trabalho da Mocidade. Possui esfera de atuação nacional.

DNTMB - Actor in Cosmos Design
Este ator participará de todos os casos de uso que envolvam as atividades do Trabalho de Membros. Possui esfera de atuação nacional.

DNTP - Actor in Cosmos Design
Este ator participará de todos os casos de uso que envolvam as atividades do Trabalho Público. Possui esfera de atuação nacional.
