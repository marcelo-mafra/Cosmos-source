Cadastro de bairos
 Anterior Acima Seguinte
A janela de cadastro de bairros é acessada através do botão "Novo Bairro", situado na janela de cadastro de logradouros. Para cadastrar um novo bairro de uma cidade, abra a janela "Cadastrar Bairro" e siga os seguintes passos:


   1. Clique no botão "Novo" para começar um novo cadastro;
   2. Digite o nome completo do bairro;
   3. No campo "Cidade", escolha na janela de pesquisa a cidade na qual o bairro se encontra; (após a escolha da cidade, são exibidos também o estado e país da cidade).
   4. Clique no botão "Salvar" para concluir a operação;
 Antes de cadastrar um novo bairro, certifique-se de que ele já não esteja cadastrado. Consulte os bairros de uma cidade analisando a grade de dados de bairros cadastrados.
