Configurador de Acesso ao Banco de Dados

O Configurador de acesso ao banco de dados oferece uma interface gráfica para manipular e armazenar as informações dos parâmetros de acesso ao banco de dados. Estas informações são usadas pelos aplicativos da suíte Cosmos, que não têm interfaces próprias para alterar tais parâmetros. O Configurador é implementado como um aplicativo do Painel de Controle tem o nome fbconf.cpl. Devido à sua especificidade, deve ser instalado apenas nos computadores onde uma ou mais aplicações servidoras estarão sendo executadas. Note-se que as aplicações servidoras não possuem qualquer mecanismo de comunicação com o Configurador, mas unicamente lêem os dados de acesso ao banco que este escreveu na Registry.

Os parâmetros manipulados pelo Configurador são descritos na tabela abaixo:
Parâmetro
Descrição
Servidor
Nome ou IP do servidor onde o banco de dados está localizado.
Arquivo de Dados
Caminho do arquivo de banco de dados ou o “alias” do banco de dados que é configurado no Firebird
Idle Timer
Indica, em minutos, quanto tempo deve ser esperado antes de desconectar conexões inativas.
Character Set
Nome do conjunto de caracteres que será usado na conexão e manipulação dos dados. Para alfabetos latinos é indicado o charset ISO8859_1.
Data de Inicialização
Data e hora em que o configurador foi inicializado.
Agente
Usuário que inicializou o Configurador.
