Opções para PDF
 Anterior  Seguinte
Os aplicativos Cosmos podem salvar qualquer um de seus relatórios ou impressos em arquivos de diferentes formatos. Para saber como salvar um relatório para um arquivo, acesse o tópico  salvando relatórios .

Quando você exporta um relatório para um arquivo no formato Adobe PDF, é possível configurar algumas informações que serão salvas neste arquivo. As seguintes informações estão disponíveis:

Informação
Descrição
Autor
Nome do autor do relatório ou impresso
Título
Título do relatório ou impresso
Assunto
Assunto abordado no relatório ou impresso
Palavras-chaves
Palavras-chaves do relatório ou impresso


Além destas informações, existem opções que podem influenciar algumas características do documento PDF gerado. A tabela abaixo descreve as opções disponíveis:

Opções
Descrição
Embutir fontes
Salva as fontes usadas no relatório no documento PDF gerado
Desabilitar hiperlinks
Se marcado, o arquivo PDF gerado não conterá os hiperlinks que eventualmente estejam presentes no relatório
Comprimir arquivo PDF
Se marcado, gera um arquivo PDF de tamanho reduzido.
