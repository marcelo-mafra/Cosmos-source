Cadastrando endereços
 Anterior  Seguinte
Para cadastrar um endereço de um centro de conferência, núcleo, sala do Trabalho Público ou de uma pessoa, você precisará conhecer o código de endereçamento postal (CEP) do logradouro onde o foco se encontra instalado ou a pessoa reside. Os aplicativos Cosmos já são distribuídos com a base nacional de cep's da Empresa de Correios e Telégrafos para facilitar o cadastro de endereços dentro do sistema.

A janela abaixo é usada para cadastrar endereços nos aplicativos Cosmos.





Tudo o que você precisa fazer é selecionar o logradouro do endereço que será cadastrado. Você faz isto seguindo os seguintes passos:
  1. Com a janela de cadastro de endereços aberta, clique no botão "Inserir" para cadastrar um novo endereço;
  2. Em seguida, clique no botão de pesquisa do campo "Logradouro";
  3. Na janela de pesquisa que se abre (janela "Localizar Logradouros"), informe o número do CEP do endereço e, em seguida, clique no botão "Pesquisar!";
  4. Se o sistema encontrar o logradouro desejado, os dados completos do logradouro serão listados na grade.
  5. Para confirmar a operação de pesquisa de logradouros e enviar os dados para a janela de cadastro de endereços, clique no botão "Selecionar".


Após selecionar o logradouro, informe os seguintes dados para completar o cadastro de endereços:
  • Número do endereço;
  • Complemento do endereço;
  • Tipo do endereço;

Finalmente, se desejar que as correspondências enviadas para o aluno, jovem do TM, membro ou pesquisador sejam remetidas para este endereço, marque a opção "Enviar correspondências para este endereço".

Observe que nem sempre é necessário informar o complemento do endereço. Tipicamente, você informaria neste campo algo como número do apartamento, número da loja ou outra informação necessária para localizar o endereço.
