Requisitos de sistema
 Anterior  Seguinte
A tabela abaixo descreve os requisitos mínimos para que os aplciativos Cosmos possam ser executados.

Requisitos
Valor(es)
Sistema Operacional
Windows 95*, Windows 98*, Windows ME, Windows 2000, Windows XP Home ou Professional, Windows 2003 ou Windows Vista
CPU
Pentium III 800 mhz ou similar
Memória
128 MB
Espaço em disco
aproximadamente 15 MB, dependendo dos aplicativos instalados. Caso o banco de dados seja instalado, cerca de 60 MB adicionais são requeridos.
 
 


* A instalação do Cosmos nos sistemas operacionais Windows 95 e Windows 98 requer a prévia atualização destes sistemas. Os pacote de atualização são fornecidas gratuitamente pela Micrososft e encontram-se no cd de instalação do Cosmos. Consulte o arquivo win9x.htm para obter informações detalhadas sobre estas atualizações.
