Cadastro de países
 Anterior  Seguinte

O cadastro de países é usado para compor um endereço. Para cadastrar um país você deve acessar a janela de cadastro de novos logradouros e, a partir dela, clicar no botão "Novo País". O cadastro de países é muito simples, requerendo que você informe apenas o nome do novo país. Contudo, certifique-se primeiro de que o país já não esteja cadastrado analisando a grade de países cadastrados.
