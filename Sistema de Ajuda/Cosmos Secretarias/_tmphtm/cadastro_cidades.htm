Cadastro de cidades
 Anterior  Seguinte
A janela de cadastro de cidades é acessada através do botão "Nova Cidade", situado na janela de cadastro de logradouros. Para cadastrar uma nova cidade, abra a janela "Cadastrar Cidades" e siga os seguintes passos:

  1. Selecione no campo "Estado" o estado no qual a cidade a ser cadastrada se encontra.
  2. Clique no botão "Novo" para começar o novo cadastro;
  3. Informe o nome da nova cidade;
  4. Clique no botão "Salvar" para concluir o cadastro;
Antes de cadastrar uma nova cidade, certifique-se de que ela já não esteja cadastrada. Consulte as cidade analisando a grade de dados de cidades cadastradas.
