Exportar dados
 Anterior  Seguinte
Os aplicativos Cosmos são capazes de exportar seus dados para alguns dos formatos de arquivos mais conhecidos. Este recurso é útil quando você precisa manipular algumas informações em um aplicativo externo, como, por exemplo, o Microsoft Excel.

Na versão 1.0 dos aplicativos Cosmos os seguintes formatos de exportação são suportados:
  • Microsoft Word (todas as versões);
  • Microsoft Excel (todas as versões);
  • Arquivos XML;
  • Arquivos HTML;
  • Texto puro (padrão ASCII).


Para exportar os dados exibidos em qualquer grade de dados exibida na janela principal, acesse a janela de exportação de dados e selecione o formato do arquivo a ser gerado. Em seguida, indique o caminho completo do arquivo a ser criado e clique no botão "Exportar!".
Seja cuidadoso ao indicar o nome do arquivo a ser gerado. Se você indicar o nome de um arquivo já existente, o mecanismo de exportação irá substituí-lo pelo novo arquivo com os dados exportados.


Se a caixa "Abrir arquivo após exportação" estiver marcada, o aplicativo Cosmos tentará abrir o arquivo gerado usando um programa que entenda o seu formato. Caso você não possua um programa adequado a esta tarefa no seu computador, a operação será abortada.
