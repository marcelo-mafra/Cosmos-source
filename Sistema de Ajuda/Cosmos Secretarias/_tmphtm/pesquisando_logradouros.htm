Pesquisando logradouros
 Anterior  Seguinte
Todos os aplicativos Cosmos diponibilizam um recurso para que você possa localizar um logradouro como base de cadastro de um endereço de um foco ou de um aluno, jovem do TM, membro ou pesquisador. Este recurso está disponibilizado através da janela "Localizar Logradouros", que é exibida abaixo.



Cadastro de novos logradouros

Além de localizar um logradouro a partir do seu Código de Endereçamento Postal (CEP), você pode executar as seguintes tarefas a partir da janela "Localizar Logradouro".

  • Cadastrar um novo país;
  • Cadastrar um novo estado, território ou província de um país;
  • Cadastrar uma nova cidade de um estdo, território ou província;
  • Cadastrar um novo bairro de uma cidade;
  • Cadastrar um novo logradouro na base de dados de Código de Endereçamento Postal;
  • Acessar o site da Empresa de Correios e Telégrafos para verificar se um determinado logradouro existe de fato.
Atenção : apenas alguns usuários possuem privilégios de cadastro de novos países, estados, cidades, bairros e logradouros. Se você precisa cadastrar qualquer uma destas informações, entre em contato com um membro da STN e solicite que a execução do cadastrado.


Antes de cadastrar um novo logradouro. você deve sempre se certificar de duas coisas importantes:
  1. Se o logradouro a ser cadastrado já não se encontra cadastrado;
  2. Se o logradouro a ser cadastrado existe na base de dados da Empresa de Correios e Telégrafos, disponibilizada na Internet

Somente quando estiver certo de que o logradouro ainda não se encontra cadastrado e existe de fato você deve cadastrá-lo. Para checar se o logradouro já se encontra cadastrado, basta digitar o CEP do logradouro e, em seguida, clicar no botão "Pesquisar!". Se houverem dados referentes ao logradouro, eles serão listados na grade de dados.


Consultando a base de dados on-line da ECT

Para checar se o logradouro existe na base de dados on-line da ECT, digite o cep do logradouro e, em seguida, clique no botão "Consultar Correios!". Você precisará dispor de uma conexão com a Internet para que esta operação seja bem-sucedida. Se o logradou existir, os dados serão retornados. Neste caso, você deveria cadastrar o novo logradouro.
Você pode consultar a base on-line de logradouros usando outros argumentos de pesquisa a partir do endereço http://www.correios.com.br/servicos/cep/cep_default.cfm
