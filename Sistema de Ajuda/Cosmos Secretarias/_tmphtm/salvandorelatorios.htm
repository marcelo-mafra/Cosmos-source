Salvando relatórios
 Anterior  Seguinte
A Central de Relatórios permite que você slave qualquer relatório ou impresso em alguns formatos de arquivos populares. Esta opção é útil quando você precisa transportar um relatório para um documento produzido por outro programa como , por exemplo, o Microsoft Word. É possível salvar qualquer relatório para os seguintes formatos de arquivo:
  • Rich Text Format (RTF) . Formato padrão que oferece suporte a texto formatados de forma complexa. Este tipo de arquivo é facilmente lido por programas de edição de texto, tais como o Microsoft Word.
  • Adobe Portable Document Format (PDF). Formato padrão para manipulação de documentos na internet. Oferece amplo suporte a impressão avançada de documentos.
  • Páginas HTML. Formato padrão para divulgação de documentos na internet. Suporta diversos recursos de formatação de texto. Pode ser lido por qualquer computador executando praticamente qualquer sistema operacional.
  • Texto puro (TXT). Formato básico de documentos de texto sem formatação.


Para salvar um relatório em um dos formatos acima descritos, siga os seguintes passos:

Selecione o relatório que se deseja salvar.
Forneça os parâmetros do relatório.
Clique no botão "Preview" para a Central de Relatórios gerar o relatório.
Na janela de previsualização do relatório, clique no botão "Salvar Relatório"
Escolha o local onde o relatório será salvo e o tipo do arquivo a ser gerado.
Clique no botão "Salvar" para salvar o relatório.
