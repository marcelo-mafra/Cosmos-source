Cadastro de estados
 Anterior  Seguinte
A janela de cadastro de estados, províncias ou territórios é acessada através do botão "Novo Estado", situado na janela de cadastro de logradouros. Para cadastrar um novo estado, província ou território, abra a janela "Cadastrar Estdos" e siga os seguintes passos:

  1. Selecione no campo "País" o país no qual o estado, província ou território a ser cadastrado se encontra.
  2. Clique no botão "Novo" para começar o novo cadastro;
  3. Informe o nome do novo estado, província ou território;
  4. Informe a sua sigla;
  5. Clique no botão "Salvar" para concluir o cadastro;
 Antes de cadastrar um novo estado, certifique-se de que ele já não esteja cadastrado. Consulte os estados de um país analisando a grade de dados de estados cadastrados.
