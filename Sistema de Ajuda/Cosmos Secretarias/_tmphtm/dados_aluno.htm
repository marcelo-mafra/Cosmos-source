Dados do aluno
 Anterior  Seguinte
A janela "Dados do Aluno" é usada para inserir as informações pessoais de um determinado aluno. Você usará esta janela toda vez que precisar alterar quaisquer informações pessoais de um aluno. Para acessar a janela "Dados do Aluno", selecione o aluno cujos dados serão editados na grade de alunos e, em seguida, clique no botão "Alterar Cadastro".

A lista abaixo enumera todos os dados existentem neste janela.

Matrícula
Número da matrícula do aluno no Lectorium Rosicrucianum. O valor deste campo é fornecido automaticamente pelo sistema.
Nome
Nome completo do aluno.
Apelido
Apelido do aluno. Use este campo apenas paraa alunos que tenham apelidos pelos quais são comumente conhecidos por muitas pessoas. Caso o aluno não tenha um apelido, o sistema usará o seu prenome como apelido.
Data de nascimento
Data de nascimento do aluno.
Sexo
Sexo do aluno.
Discipulado
Discipulado no qual o aluno se encontra. Este campo é apenas informativo.
Identidade
Documento de identificação do aluno. Documentos de identificação típicos são RG's, documentos de cadastro em instituição classista ou documentos emitidos por instituições militares.
Expedidor
Sigla da órgão expedidor do documento de identificação.
Estado civil
Estado civil do aluno.
Nacionalidade
Nacionalidade do aluno. Se o aluno possuir mais de uma nacionalidade, insira a que julgar mais importante constar no cadastro.
Profissão
Profissão do aluno. A profissão deve ser selecionada da lista de profissões disponíveis. Observe que se o aluno já estiver aposentado, deve-se inserir de qualquer forma a profissão que exercia antes da aposentaria. Se v não encontrar a profissão do aluno na lista de profissões, deverá entrar em contato com um membro da STN e solicitar que ele cadastre a nova profissão para que você possa atribiu a nova profissão ao aluno.
Aluno éxtra núcleo?
Marque esta opção caso o aluno seja extra-núcleo.
Pertence ao GJA?
Marque esta opção caso o aluno seja membro do Grupo de Jovens Alunos.
Aluno é freqüente nas atividades?
Marque esta opção caso o aluno seja muito freqüente nas atividades promovidas pelo Lectorium Rosicrucianum. Se marcada, esta opção fará com que o aluno receba presença como valor padrão toda vez que você iniciar o levantamento de freqüência de uma atividade.
Aluno é ligado ao Rozenhof?
Marque esta opção caso o aluno seja vinculado ao Rozenhof.
Aluno é mentor da Escola Interna?
Marque esta opção caso o aluno seja um mentor da Escola Interna, ainda que atue neste trabalho de forma ocasional.
Aluno é licenciado?
Marque esta opção caso o aluno seja licenciado.
Aluno titular
Informe neste campo qual o aluno que é o titular do aluno cujos dados estão sendo editados. Se este não possuir um aluno titular, deixe o campo vazio. Se ele possuia um aluno titular mas agora já não o possui. clique no botão "Retirar Titular" para desvinculá-lo de seu antigo titular.
Mensalidade
Informe o valor da mensalidade a ser paga pelo aluno.
Taxa de conferência
Informe o valor da taxa de conferência que o aluno paga no centro de conferências ao qual está vinculado.
Notas
Insira observações relevantes referentes aos dados cadastrais do aluno neste campo. Não insira informações sobre o seu histórico discipular, endereço ou meios de contato neste campo.
Foto do aluno
Para inserir uma foto do aluno, clique com o botão direito do mouse no campo reservado para a foto do aluno e, em seguida, escolha menu "Procurar Foto". Para excluir a foto existente, escolha o menu "Apagar Foto".
