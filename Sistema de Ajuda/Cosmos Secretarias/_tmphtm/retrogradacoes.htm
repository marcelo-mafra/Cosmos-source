Retrogradaçoes
 Anterior  Seguinte
Uma retrogradação é um ato no qual um aluno é transferido para um discipulado antecessor àquele em que se encontra no momento em que a retrogradação é feita.

Para registrar uma retrogradação de uma aluno, siga os seguintes passos:

  1. Abra a janela "Alunos do Lectorium". (Acesse os menus Secretarias/Aluno do Lectorium);
  2. Selecione a grade de alunos o aluno que será retrogradado.
  3. Na barra de ferramentas da janela aberta. clique no botão "Retrogradar Aluno".


A janela abaixo será exibida:




Nesta janela, você vê as seguintes informações:

Foco
Indica a secretaria de qual foco está aberta.
Aluno
Nome do aluno que será retrogradado
Retrogradar Para
Informe para qual discipulado o aluno será retrogradado.
Data
Informe a data em que o aluno foi retrogradado.
Motivo
Informe o motivo da retrogradação.
Descrição sumário
Se necessário insira uma descrição sumária da retrogradação
Notas
Opcionalmente, insira quaisquer notas sobre a operação.


Quando estas informações estiverem preenchidas, clique no botão "Retorgradar" para concluir a operação. Confirme a execução da operação para retrogradar o aluno.

O sistema irá automaticamente inserir no histórico discipular do aluno o evento da retrogradação e passá-lo para o novo discipulado indicado.
Apenas alunos podem ser retrogradados.
