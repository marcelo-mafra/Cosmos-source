Modelo de licenciamento
 Anterior  Seguinte
Os programas do Cosmos precisam ser registrados para que possam ser usados. O modelo de licenciamento é definido pelo Presidium, mas envolve sempre o prévio registro dos aplicativos como condição para que eles possam ser liberados para uso.

Cada um dos aplicativos que fazem parte da suíte Cosmos precisa obter uma licença de uso distinta. Além disto, as licenças são sempre destinadas ao uso de um aplicativo Cosmos em um determinado computador. Isto quer dizer que se você decidir reinstalar o Cosmos em um novo computador, terá de solicitar uma nova licença do produto. Desta forma, se você instalar o pacote completo de aplicativos Cosmos em um único computador, terá de solicitar 4 licenças de uso:
  • 1 licença para o Cosmo Gestor de Focos
  • 1 licença para o Cosmos Gestor de Secretarias
  • 1 licença para o Cosmos Gestor Financeiro
  • 1 licença para o Cosmos Gestor de Conferências

O processo de registro de cada um destes aplicativos é feito de forma separada.


Do registro

Para registrar qualquer um dos aplicativos da suíte Cosmos, é necessário seguir o mesmo processo. Um resumo do processo de registro é descrito abaixo:

1) Solicitação ao Presidium de uma licença de uso.
2) Se a licença for concedida, será recebido um arquivo de licença de uso.
3) Registro do aplicativo com o arquivo de licença de uso recebido.

Todas as vezes que você inicializa uma aplicativo Cosmos, é checado se o mesmo se encontra registrado. Caso o registro ainda não tenha sido feito, a janela de registro será exibida. Toda a operação de registro é feita através desta janela. Veja abaixo uma imagem da janela de registro.




Após digitar os dados requeridos (em negrito), clique no botão "Gerar Arquivo" para que as informações de registro sejam geradas e salvas em um arquivo de requisição de licença. Este arquivo deve ser enviado para o email cni@lectoriumrosicrucianum.org.br.
Feche o aplicativo em seguida, pois será necessário aguardar a aprovação do pedido de licença. Caso isto aconteça, você receberá, no email informado na tela de registro, um arquivo contendo os dados do registro da licença.

Para concluir o registro, abra novamente, no mesmo computador que gerou o arquivo de licença, o aplicativo a ser registrado. Quando a janela de registro for exibida, clique no botão "Registrar!", selecione o arquivo de licença recebido (não o arquivo de requisição gerado por você!) e aguarde até que a operação de registro se conclua. Se a operação de registro for completada com sucesso, o aplicativo estará registrado e estará liberado para o uso.
