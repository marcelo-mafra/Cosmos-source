Tools Manager
 Anterior Acima Seguinte
O Tools Manager é uma ferramenta simples que objetiva permitir que o usuário dos aplicativos Cosmos possam acessar de forma rápida as ferramentas externas que mais utiliza. Para executar qualquer ferramenta externa, basta selecioná-la na grade de ferramentas e clicar no botão "Executar".


Cadastrando novas ferramentas externas

Para cadastrar uma nova ferramenta externa no Tools Manager, clique no botão "Novo..." Na janela "Nova Ferramenta", forneça as seguintes informações:

Ferramenta
Nome de registro da nova ferramenta no Tools Manager.
Arquivo/comando
Caminho completo do arquivo ou comando a ser executado.
