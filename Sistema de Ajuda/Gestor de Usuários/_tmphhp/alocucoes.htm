Alocuções
 Anterior Acima Seguinte
O Cosmos Gestor de Secretarias provê um cadastro geral de alocuções que são proferidas nas atividades promovidas pelos focos do campo de trabalho brasileiro. O propósito deste cadastro é fornecer um conjunto nacional comum de alocuções, disponíveis para todos os focos brasileiros.

Desde que possua direitos para tanto, você pode cadastrar uma referência para qualquer alocução existente, de forma que um usuário do Gestor de Secretarias poderá agendar uma atividade (um serviço templário, por exemplo), informando que nela será proferida a alocução cadastrada por você. O Cosmos Gestor de Secretarias permite cadastrar referências a alocuções dos seguintes campos de trabalho:

  • Lectorium Rosicrucianum;
  • Trabalho da Mocidade;
  • Trabalho de Membros;
  • Trabalho Público.


Na grade de alocuções são inicialmente listadas as alocuções recentemente proferidas. Usando o menu de contexto é possível, contudo, listar apenas as alocuções que atendam uma das seguintes condições:

Alocuções recentes
 
Alocuções inéditas
 
Novas alocuções
