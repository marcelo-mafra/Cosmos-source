Gerenciador de janelas
 Anterior Acima Seguinte
O Gerenciador de Janelas é um recurso que possibilita o exame de todas as janelas abertas por um usuário. Todos os aplicativos da suíte Cosmos disponibilizam O Gerenciador de Janelas. Nesta ferramenta você pode executar as seguintes tarefas:

  • Listar todas as janelas abertas na sua seção de trabalho;
  • Examinar informações detalhadas sobre cada uma das janelas abertas;
  • Ativar uma janela;
  • Fechar uma janela.

O Gerenciador de Janelas permite a visualização das informações sobre as janelas abertas de duas formas:
  • Modo de ícones : lista as janelas de forma sumária. Cada janela aberta é representada pelo seu ícone.
  • Mode de listagem: listas as janelas usando os seus respectivos ícones, nomes e uma descrição simples dos recursos oferecidos na janela.

Para alternar entre os modos de visualização, clique nos botões dispostos lateralmente no Gerenciador de Janelas. A imagem abaixo descreve detalhadamente os recursos desta ferramenta.


Para acessar o Gerenciador de Janelas, acesse os menus "Janelas>Janelas..." ou clique no botão  da barra de ferramentas.
