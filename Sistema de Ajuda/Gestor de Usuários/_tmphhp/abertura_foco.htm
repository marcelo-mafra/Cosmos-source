Abertura de foco
 Anterior Acima Seguinte
A janela "Abertura de Foco" é utilizada toda vez que você deseja escolher com qual foco irá trabalhar. Como o Gestor de Secretarias e o Gestor Financeiro apenas permite que você trabalhe com apenas um foco por vez, você acessará esta janela sempre que desejar escolher o foco com o qual vai trabalhar.

Nesta janela é exibida uma mensagem de boas-vindas ao usuário atualmente logado e, logo abaixo, uma lista com todos os focos com os este usuário está autorizado a trabalhar. O conjunto destes focos pode variar muito para cada usuário, e depende exclusivamente dos privilégios de acesso concedidos a eles pelos membros do Presidium.

Para abrir o foco, basta selecioná-lo na lista de focos e clicar no botão "OK". O mesmo resultado é obtido dando-se um duplo clique sobre o nome do foco que se deseja abrir.





Se você clicar no botão "Logoff", a conexão estabelecida será desfeita.
