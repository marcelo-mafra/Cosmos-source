DCOM - definição

DCOM (Distributed component object model) é uma tecnologia da Microsoft para criação de componentes de software distribuídos em computadores interligados em rede. O DCOM é uma extensão do COM (também da Microsoft) para a comunicação entre objetos em sistemas distribuídos.


Sistema Cosmos e o DCOM

Os aplicativos do sistema Cosmos podem utilizar o DCOM para estabelecer a comunicação e tráfego de dados entre si. De forma simplificada, o DCOM é utilizado como mecanismo (protocolo) através do qual alguns tipos de aplicações, chamadas de "aplicações clientes", comunicam-se com outros tipos de aplicações denominadas "aplicações servidoras de dados". Esta comunicação acontece se todos os aplicativos Cosmos (aplicações clientes e aplicações servidoras de dados) estiverem instalados em um mesmo computador ou se estiverem instalados em computadores diferentes, em uma rede.

No primeiro caso (todos os aplicativos instalados em um mesmo computador), a configuração do DCOM é feita de forma totalmente automática pelo instalador do Sistema Cosmos. Contudo, quando as aplicações clientes são instaladas em computadores diferentes das servidoras de dados, esta configuração do DCOM não pode ser feita de forma automática pelo instalador. Será necessário, então, acessar a ferramenta de configuração do DCOM, distribuída com o Microsoft Windows.
